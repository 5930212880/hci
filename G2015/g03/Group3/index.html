About Project

  • Helmet Intelligent

    HOW?

    เป็นหมวกกันน็อคที่มากความสามารถ โดยไม่ได้เป็นแค่เครื่องป้องกันอันตรายที่เกิดกับศีรษะ แต่ยังเสริมฟีเจอร์ต่างๆที่น่าสนใจ และที่สำคัญสามารถตามหาได้หากหายหรือโดนขโมย

  • Application Connecting

    HOW?

    หมวกกันน็อคจะเชื่อมต่อทุกฟังก์ชันได้ด้วยโทรศัพท์มือถือ เพราะในโทรศัพท์มือถือจะต้องโหลดแอพพลิเคชันที่ชื่อว่า HELSIT สำหรับหมวกกันน็อคโดยตรง เพียงแค่แอพพลิเคชันเดียวครบวงจร

  • Bluetooths Connect

    How?

    มีหูฟังบลูธูทในหมวกกันน๊อคและสามารถเชื่อมต่อกับโทรศัพท์มือถือโดยผ่านแอปพลิเคชันได้ อีกทั้งยังสามารถรับโทรศัพท์ขณะขับขี้รถมอเตอร์ไชค์ได้ โดยไม่ต้องยกโทรศัพท์ขึ้นมา คือ รับได้โดยแตะที่หมวกกันน็อคด้านใดด้านหนึ่ง ที่มีหูฟังบลูธูท

  • Global Wi-fi

    "How?"

    มี wifi-hotspot ในตัว เพียงแค่เข้าไปที่การตั้งค่าในแอปพลิเคชัน ค่าบริการ wifi hotspot นั้นยังคงคิดค่าบริการแยกจากหมวกกันน็อค กล่าวคือ ขึ้นอยู่กับว่าลูกค้าจะใช้บริการเครือข่ายใดหรือใช้บริการอินเตอร์เน็ตโปรโมชันไหนของเครือข่ายนั้นๆ

  • Distanc Alarm

    How?

    จะส่งสัญญาณเตือนเข้าไปในโทรศัพท์ของผู้ใช้เมื่อเขาออกห่างจากหมวกกันน็อคเป็นระยะทางที่มากๆ เนื่องจากหากตั้งค่าเปิด Lost Mode ไว้ ทุกอย่างจะเตือนโดยอัตโนมัติ ซึ่งผู้ใช้จะสามารถเปิด Find My Helmet เพื่อตามหาหมวกกันน็อคได้ในทันที

  • Camera

    "How?"

    มี wifi-hotspot ในตัว เพียงแค่เข้าไปที่การตั้งค่าในแอปพลิเคชัน ค่าบริการ wifi hotspot นั้นยังคงคิดค่าบริการแยกจากหมวกกันน็อค กล่าวคือ ขึ้นอยู่กับว่าลูกค้าจะใช้บริการเครือข่ายใดหรือใช้บริการอินเตอร์เน็ตโปรโมชันไหนของเครือข่ายนั้นๆ

  • GPS Lost Mode

    How?

    มีระบบ GPS ซึ่งช่วยป้องกันการสูญหายหรือถูกลักขโมยได้ หากเปิด Lost Mode ไว้ ระบบจะทำการแจ้งเตือนโดยอัตโนมัติ อีกทั้งยังสามารถใช้ในการติดตามแบบเรียลไทม์ได้ทั่วโลก และที่สำคัญ ในแอปพลิเคชันยังมี GPS ค้นหาจุดหมายที่เราต้องการไป และยังนำทางไปถึงจุดมุ่งหมายอีกด้วย

  • Charging Battery

    "How?"

    เนื่องจากเป็นหมวกกันน็อคอัจฉริยะ ที่เชื่อมต่อกับระบบต่างๆและทำให้ใช้พลังงาน จึงจำเป็นต้องมีการชาร์จเพื่อให้หมวกกันน็อครับพลังงาน และสามารถนำพลังงานไปใช้ในระบบ ได้อย่างมีประสิทธิภาพ
